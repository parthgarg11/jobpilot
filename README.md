# JobPilot 🧭

A full-stack job application tracker built with Spring Boot and React, 
featuring AI-powered JD analysis, cover letter generation, and interview 
prep — backed by Google Gemini.

## Tech Stack
- **Backend:** Java 21, Spring Boot 3.2, Spring Security, JWT
- **Database:** PostgreSQL
- **Frontend:** React 18, Vite, Tailwind CSS
- **AI:** Google Gemini API

## Features
- Track job applications with status pipeline
- AI-powered job description analyzer
- Cover letter generator
- Interview prep question generator  
- Automated follow-up nudge agent
